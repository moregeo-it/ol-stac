STAC Item crossing the antimeridian

stac15 stacitem9 antimeridian1 footprint1

Rendering a STAC Item whose geometry crosses the antimeridian (180° longitude).

The footprint of this STAC Item crosses the antimeridian (180° longitude). Such geometries are split into multi-geometries so that they render correctly on the map instead of wrapping around the whole globe. The example data is a thunderstorm warning for Mahia, New Zealand, taken from stac-browser issue #736.

const background = new TileLayer({
  source: new OSM(),
});

const map = new Map({
  target: 'map',
  layers: [background, layer],
  view: new View({
    center: [0, 0],
    zoom: 0,
    // The fitted extent of an antimeridian-crossing bbox extends beyond +180°,
    // so multiWorld must be enabled to avoid clamping the wrapped portion.
    multiWorld: true,
  }),
});
layer.on('sourceready', () => {
  const view = map.getView();
  view.fit(layer.getExtent(), {padding: [50, 50, 50, 50]});
});
